# behavysis_pipeline

[Documentation](https://tlee08.github.io/behavysis_pipeline/)

Core functionalities shared across the Behavysis suite.

## Installation

### Dev installation

```bash
conda env create -f conda_env.yaml
conda activate behavysis_pipeline_env
pip install poetry
poetry install
```

### User installation

```bash
conda env create -f conda_env.yaml
```
